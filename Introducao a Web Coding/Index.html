2. Parágrafos

Os parágrafos são como contar uma história. É onde as palavras vivem e brincam!
Vamos imaginar que estamos contando uma grande aventura.
Primeiramente, começamos com uma introdução sobre nossos heróis.

Depois, descrevemos os desafios e as aventuras que eles enfrentam.

No final, contamos como eles venceram e aprenderam coisas novas.
Cada parágrafo é como um pedaço dessa história, cheio de emoções e surpresas.

Vamos usar os parágrafos para organizar todas essas partes, assim como fazemos ao montar um quebra-cabeça!
